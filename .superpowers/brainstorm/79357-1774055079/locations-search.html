Key Features

  • Free text search across all fields + individual field filters
  • Status filter (Verified, Needs Review, Rejected)
  • Confidence score color-coded: green (70+), yellow (30-69), red (<30)
  • Pagination with configurable page size (20/50/100)
  • URL-based state — search params in URL for shareability
  • Rejected items shown dimmed with strikethrough (toggle via filter)
  • Click any row to navigate to /locations/[id]
